# Amazon Product Search Skill

Amazon Product Search skill helps users search and analyze products on Amazon across different marketplaces.

## Description

This skill provides comprehensive Amazon product search capabilities including:
- Product search by keywords
- Price comparison across marketplaces
- Product details and specifications
- Review analysis
- Sales rank tracking
- Category browsing

## Capabilities

### Search Products
- Search Amazon products by keyword, ASIN, or category
- Filter by price range, brand, ratings, and more
- Sort by relevance, price, reviews, etc.

### Product Analysis
- Get detailed product information
- Analyze pricing history
- Check seller information
- Review product images and descriptions

### Marketplace Support
- Amazon US (amazon.com)
- Amazon UK (amazon.co.uk)
- Amazon DE (amazon.de)
- Amazon JP (amazon.co.jp)
- And more...

## Usage Examples

```
"Search for wireless earbuds under $50 on Amazon US"
"Find products with ASIN B08N5WRWNW"
"Compare prices for iPhone 15 across Amazon marketplaces"
"Show top-rated coffee makers on Amazon"
```

## Configuration

Requires Sorftime API integration for Amazon data access.
