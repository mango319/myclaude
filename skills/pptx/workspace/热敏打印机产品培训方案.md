### 场景三要素法

```
场景 = WHO + WHEN + WHAT
```

## 3.4 需求管理与优先级

### RICE模型实战

```
RICE = (Reach × Impact × Confidence) ÷ Effort

Reach：触达多少用户？
Impact：对用户价值多大？（1-3分）
Confidence：信心多大？（1-100%）
Effort：需要多少工作量？（人月）

实战案例：

【需求A】批量导入Excel打印价签
Reach：80%商超客户会用
Impact：3分（大幅提升效率）
Confidence：90%（有现成方案）
Effort：2人月
RICE = (80 × 3 × 0.9) ÷ 2 = 108

【需求B】支持语音打印标签
Reach：10%家庭用户会用
Impact：2分（锦上添花）
Confidence：60%（技术有风险）
Effort：4人月
RICE = (10 × 2 × 0.6) ÷ 4 = 3

结论：优先做需求A
```

### KANO模型实战

```
需求分类：
├─ 基本型需求：必须有，做了用户不说好，不做骂
├─ 期望型需求：做得越好用户越满意
├─ 兴奋型需求：做了用户惊喜，不做无所谓
└─ 反向需求：做了反而减分

热敏打印机需求分类：

基本型（必须做好）：
✓ 能打印、不卡纸
✓ 打印清晰
✓ 操作简单

期望型（持续优化）：
◯ 打印速度
◯ 连接稳定性
◯ 耗材成本

兴奋型（打造爆款）：
★ APP模板库
★ AI图片识别打印
★ 云端同步

反向（千万别做）：
✗ 强制广告
✗ 复杂的会员系统
✗ 闭锁的耗材体系
```

## 3.5 产品定位与差异化

### 产品定位陈述模板

```
公式：
为 [目标用户]
在 [使用场景]
提供 [核心价值]
不同于 [竞品/替代方案]
我们 [差异化优势]

示例：
【便携式标签打印机】
为 小零售店主、家庭用户
在 日常标签打印、收纳整理场景
提供 便携、易用、经济的标签打印方案
不同于 手写标签、大型打印设备
我们 体积小巧、操作简单、价格亲民

【工业条码打印机】
为 工厂、制造业用户
在生产流水线、产品追溯场景
提供 高耐用、高稳定性的标贴打印方案
不同于 普通商用打印机
我们 支持24小时连续运行、工业级防护、超长质保
```

### 差异化策略矩阵

```
差异化维度：

【性能差异化】
├─ 打印速度：160mm/s vs 竞品120mm/s
├─ 打印精度：203dpi vs 竞品180dpi
└─ 打印宽度：支持4寸 vs 竞品最大3寸

【成本差异化】
├─ 设备价格：同配置低10-20%
├─ 耗材成本：热敏纸便宜15%
└─ 维护成本：模块化设计，维修便宜

【体验差异化】
├─ 易用性：一键换纸 vs 竞品需打开盖子
├─ 智能化：APP远程打印 vs 竞品仅USB
└─ 服务：24小时售后 vs 竞品48小时

【场景差异化】
├─ 细分场景：奶茶店专用模板
├─ 行业定制：医疗腕带专用模式
└─ 生态整合：对接外卖平台
```

## 4.6 常见问题与解决方案

### 产品-销售协作常见痛点

| 痛点 | 表现 | 根本原因 | 解决方案 |
|------|------|----------|----------|
| **销售不懂产品** | 参数记不住、场景匹配错 | 培训不到位、工具不好用 | 简化培训+优化工具 |
| **产品不懂市场** | 功能做出来没人买 | 缺少市场反馈、闭门造车 | 建立反馈机制、定期沟通 |
| **信息不同步** | 产品变了销售不知道 | 沟通渠道不畅 | 建立信息同步机制 |
| **反馈无响应** | 销售反馈了没下文 | 缺少反馈处理流程 | 建立分级处理机制 |
| **目标不一致** | 产品要体验、销售要便宜 | KPI不统一 | 对齐目标、建立共同指标 |
| **互相抱怨** | 销售说产品不行、产品说销售不行 | 缺少理解和信任 | 增加互动、共建共创 |
